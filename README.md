# actuarial-reserve-calculator
Actuarial reserve calculation software developed as a bachelor thesis project in Financial Engineering


# Features
- Actuarial function calculator
- Mortality table management
- Actuarial equation solver
- Mathematical reserves
- IBNR: Chain Ladder, Bornhuetter-Ferguson and GLM
- RBNS calculations
- UPR calculations
- Data file management
- Tkinter GUI


# How to Run
1. Download `main.py` and the `Mortality Tables` folder
2. Optionally download example `Claim Files` and `Policy Files`
3. Place the `Mortality Tables` folder in the same directory as `main.py`
4. Open `main.py` in any Python IDE and run it. Install missing libraries if necessary

 
# Note
When running the program in Spyder IDE, the application may occasionally crash on first launch because of an IDE-related issue. Running the program again resolves the problem
